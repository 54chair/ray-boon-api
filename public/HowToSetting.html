OpenAI API key 設定方法

RAY-BOON では、例文生成に OpenAI の ChatGPT を使用しています。
例文生成をするためには、OpenAI の API key が必要になります。
以下の手順で取得してください。

1. OpenAI にログイン

https://openai.com/ にアクセスし、 ログインします。
アカウントを持っていない場合は、新規登録を行ってください。

2. API key を取得

https://platform.openai.com/account/api-keys にアクセスします。
以下のような画面が表示されます。

Create new secret key を クリックします。 タップします。
API key の名前（自由に決めてください）を入力し、Create secret key をクリックします。タップします。
API key が表示されるので、メモ帳などに保存してください。

(API key は他人に知られないようにしてください。)

以上で、OpenAI API key の取得は完了です。

3. RAY-BOON に API key を入力

取得した API key を、RAY-BOONアプリの
設定→APIキー設定を入力
から入力してください。

これで、RAY-BOON で例文生成ができるようになりました。
